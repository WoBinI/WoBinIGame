# WoBinI Game
### Angular2 based StreetView Game
> A sample project from students of FH Kufstein

#### Run application
1. Clone Repository `git clone https://github.com/WoBinI/WoBinIGame.git`
1. cd WoBinIGame
1. Install [Node.js](https://nodejs.org/en/)
1. Run `npm install`
1. Run `npm start`

Enjoy the game
